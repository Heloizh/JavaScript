# Simple Chatbot

## Descrição

Este projeto é um chatbot simples desenvolvido usando HTML, CSS e JavaScript. Ele simula um chatbot básico que pode responder a algumas perguntas predefinidas. O objetivo deste projeto é demonstrar como criar uma interface de bate-papo interativa e responsiva.

## Funcionalidades

- **Interface de Bate-Papo**: Uma área de chat onde o usuário pode digitar suas perguntas.
- **Respostas Predefinidas**: O chatbot responde com respostas predefinidas baseadas em palavras-chave detectadas na entrada do usuário.
- **Desempenho Dinâmico**: A interface atualiza dinamicamente para exibir novas mensagens do usuário e respostas do chatbot.

## Estrutura de Arquivos

- `index.html`: Contém a estrutura HTML do chatbot.
- `style.css`: Contém os estilos CSS para o layout e design do chatbot.
- `script.js`: Contém a lógica do chatbot utilizando JavaScript.

## Como Usar

1. Abra o arquivo `index.html` em um navegador.
2. Digite uma mensagem na área de entrada de texto.
3. Pressione Enter ou clique no botão de envio para ver a resposta do chatbot.
4. Continue a interação conforme necessário.
